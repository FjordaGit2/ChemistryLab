Chemistry Lab With Neurofeedback

Aims:
To understand users ability to focus on a switched attention task, their visual memory, their response strategy, and motor impulsivity.

Groups of participants:

-Controls

-People with ADHD/ADHD traits

Task Details:
In this project, users are immersed in a virtual chemistry lab with interactive equipment. A periodic table is displayed in front of them, and selecting an element triggers an animated 3D model of the corresponding atom. Participants are guided by a virtual robot to complete several chemistry experiments. The interactivity of the equipment is dynamically controlled by the user's attention level, measured via EEG: if attention falls below 50%, the equipment becomes non-interactable, and users can only proceed once their attention exceeds this threshold. This VR-based educational application was designed to explore whether immersive environments combined with neurofeedback can enhance learning outcomes in classroom and distance learning contexts. The task evaluates sustained attention, alternating and visual attention, visual memory, and response strategies.
